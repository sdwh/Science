1747 - What are the common half-reaction formulas for oxidizing and reducing agents?

When dealing with half-reactions in chemistry, we need to remember the following common formulas for oxidizing and reducing agents:

  1. Oxidizing Agents:
  2. O2 + 4H+ + 4e- → 2H2O
  3. MnO4- + 8H+ + 5e- → Mn2+ + 4H2O
  4. Cr2O72- + 14H+ + 6e- → 2Cr3+ + 7H2O

  5. NO3- + 4H+ + 3e- → NO + 2H2O

  6. Reducing Agents:

  7. Zn → Zn2+ + 2e-
  8. Mg → Mg2+ + 2e-
  9. Fe2+ → Fe3+ + 1e-
  10. H2 → 2H+ + 2e-
