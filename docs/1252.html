Science Common Sense

1252 - What can cause an object to move or rotate?

There are two main things that can cause an object to move or rotate:

  1. A push or a pull (called force): When you push or pull something, it can make the object move.
  2. A turn or a twist (called torque): When you turn or twist something, it can make the object rotate.

For example, when you push a ball, it moves (because of a push force). When you turn a door handle, it opens (because of a twist torque).
